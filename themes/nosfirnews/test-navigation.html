Teste de Layouts de Navegação

Esta página demonstra os diferentes layouts de navegação disponíveis no tema NosfirNews. Use os controles acima para testar diferentes configurações.

O sistema de navegação inclui:

  • Posições flexíveis: Abaixo do header, inline com header, à direita do logo, próximo ao logo, centro e direita
  • Alinhamentos: Esquerda, centro e direita
  • Estilos visuais: Padrão, minimal, boxed e underlined
  • Responsividade: Adaptação automática para dispositivos móveis
  • Acessibilidade: Navegação por teclado e leitores de tela

O layout é totalmente responsivo e se adapta automaticamente a diferentes tamanhos de tela, garantindo uma experiência consistente em todos os dispositivos.
